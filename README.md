# ThoughtWorks Cruise F2E HomeWork

## 命令  
#### 0. 安装
```bash
    npm install or yarn install  
```  

#### 1. 运行开发环境  
```bash
    npm start or yarn start  
```

#### 2. 运行单元测试  
> 第一次如果报错，为暂未生成快照，再执行一次即可。  
```bash
    npm test or yarn test  
```

#### 3. 输出生产环境  
```bash  
    npm run build --report  or yarn run build --report  
```
